## Démarrage local

### Prérequis

- Node.js 18+
- npm

### Installation

```bash
# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev

# Construire pour la production
npm run build

# Prévisualiser la version production
npm run preview
```
